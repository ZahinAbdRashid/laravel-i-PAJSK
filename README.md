# i-PAJSK System

This project is a web-based co-curricular assessment system developed using Laravel.

## Features
- Role-based authentication (Admin, Teacher, Student)
- PAJSK score management
- Dashboard system

## Tech Stack
- Laravel
- Tailwind CSS
- MySQL
